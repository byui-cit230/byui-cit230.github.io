Overview

In this lesson's application assignment we will work on the home page (index.html) for the Weather Site. This home page will display general information about our three towns. The assignment follows the activity that you completed in the Prepare module of this lesson to populate the content areas with an outside JSON data source. This data source contains information such as the average rainfall, current population, and a list of town events.

Instructions

  1. Create the weather site home page (index.html) using the weather site planning document and your web site planning document as a guide. Store this page in the lesson-9 folder of your assignments directory and be sure to include the supportive folders and files that support this weather site assignment update.
  2. Create a wireframe sketch of the home page for the large and small views using any viable software tool. You will be responsible for the design and content of the main area of the home page. Post this sketch or its URL to the Developer Forum for peer review and discussion.

    Helpful Tip

    Wireframe.cc is a free, online tool that allows you to save and share your URL.

  3. Your home page should contain a section displaying all three towns with the provided data and include images displayed in three columns in the large view and a single column in the small view. Use CSS flex or CSS Grid to achieve the layout.

Display the individual town data with the motto, year founded, population, and annual rainfall from the JSON file on your home page. Use the lesson's activity as your guide to read in the data and display on the page.
